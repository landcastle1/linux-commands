# 조사 대상 명령어
- 'top'
- 'ps'
- 'jobs'
- 'kill'

---

## top

### 개요  
'top' 명령어는 시스템의 CPU·메모리·프로세스 사용 상태를 실시간으로 보여주는 모니터링 도구이다. 성능 진단 및 실행 중인 프로세스 확인에 자주 사용된다.

### 주요 정보 항목

항목    	의미
PID	      프로세스 ID
USER	    실행한 사용자
%CPU	    CPU 사용량
%MEM	    메모리 사용량
TIME+	    실행 시간

### 유용한 단축키

키	기능
h	  도움말 표시
M	  메모리 사용 기준 정렬
P	  CPU 사용 기준 정렬
k	  프로세스 종료
q	  top 종료

---

## ps

### 개요
'ps'는 현재 실행 중인 프로세스를 스냅샷 형태로 출력하는 명령이다. top과 달리 실시간 갱신되지 않는다.

### 주요 옵션
옵션	의미
a	    다른 사용자 포함 전체 프로세스 표시
u    	프로세스 소유자 정보 포함
x	    터미널과 연결되지 않은 프로세스 포함
-ef	  전체 프로세스를 풀 포맷으로 출력

---

## jobs

### 개요
jobs 명령어는 현재 셸 세션에서 실행 중인 백그라운드 작업 목록을 확인할 때 사용된다.

### 출력 상태 종류
상태	    의미
Running	  작업 실행 중
Stopped	  일시 정지된 작업
Done	    작업 완료

---

## kill

### 개요
kill 명령어는 실행 중인 프로세스에 signal을 보내 종료 또는 제어하는 명령이다.

### 주요 옵션
옵션	  Signal	    의미
-15	    SIGTERM    	정상 종료 요청 (기본)
-9	    SIGKILL	    강제 종료
-1	    SIGHUP	    재시작 요청

---

## 명령어 비교 요약표

명령어     역할	                                      실시간 여부
top	       시스템 및 프로세스 상태 실시간 모니터링	    ✔ 실시간
ps	       특정 시점 프로세스 목록 출력	              ✘ 스냅샷
jobs	     현재 셸의 백그라운드 작업 확인	            ✔ 제한적
kill	     실행 중인 프로세스 종료 또는 제어	          ✘ 제어용
